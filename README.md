# Awesome-ReciteEnglishWord-Ebbinghaus-Forgetting-Curve
本项目主要是根据英文单词(国际音标,词性,中英释义)数据集及各年龄阶段英语考试试题数据集，根据个人账号的记忆记录按照艾宾浩斯遗忘曲线构建每日英语测试试题，以提高每个人的英语单词记忆效率，以战代练。

1.每天10个单词，第一页为该次需要背诵的单词
2.第二题为根据单词英文写汉语意思或根据汉语意思写英文单词，顺序按照艾宾浩斯遗忘曲线
3.第三题为阅读理解，用算法搜索实体库中包含该次10个单词的阅读理解题目，并在题目中加粗背诵任务中的单词
4.第四题为选词填空
5.第五题为选择句子填空
6.完成试卷后根据标准答案统计每一道题目的对错情况，录入该系统，以便下次单词背诵任务的单词选取
